Unbeatable AI Search Engine System - Advanced Plan

System Overview

A multi-agent, self-questioning AI search engine that reasons, searches, summarizes, and presents answers from multiple trusted sources in a fully automated pipeline.

Process Flow

  1. User Input → Natural language question
  2. Query Classification → Detects type and topic
  3. Sub-question Generation → Breaks query into related parts
  4. Agent Assignment → Researcher, Summarizer, Critic
  5. Parallel Source Search → Multiple databases via SearxNG
  6. Content Extraction → Scrape + clean info
  7. Answer Composition → Summarize and cite
  8. Critique + Merge → Review and unify response
  9. Output Answer → With breakdown + sources

Combined Features

  • Multi-source scraping (Wikipedia, Reddit, GitHub, StackOverflow)
  • Self-questioning engine (Tree of Thought)
  • Multi-agent collaboration (from MindSearch)
  • Cited summary with inline links
  • Critique module to detect hallucinations
  • Adaptive search mode (e.g., academic, technical, opinion)

Tech Stack (Free / OSS)

  • React + TypeScript + Tailwind – Frontend
  • Vercel Functions – Serverless API layer
  • SearxNG – Meta search engine
  • OpenRouter API – LLM access (Claude, Mixtral, etc.)

Frontend UX Highlights

  • Smart Input Box + Suggestions
  • Breakdown View with Expand/Collapse
  • Citation Graph Visual
  • Retry & Regenerate per sub-answer
  • Dark Mode and Responsive Design
